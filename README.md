# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

# Display icons

To display icons, please install a nerd-fonts first.

Press `Command + ,` to open iTerm2 settings, and then select a nerd-fonts at `Profiles > Text > Font`.

# Meta key

Open iTerm2 settings, go to `Profiles > Keys > Left Option Key / Right Option Key` and select the preference of direction of option key to `ESC+`
